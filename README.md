# 保研领航员 Baoyan Pilot

面向大学生的 AI 保研规划助手第一版，使用 React + Vite + Tailwind CSS 构建。

## 页面

- 首页：产品介绍、核心功能、使用流程、开始使用入口和免责声明。
- 保研画像评估页：左侧填写个人信息，右侧生成模拟画像分析和 30 天行动计划。
- 院校资料库页：按院校层次、专业方向、申请类型筛选院校卡片。
- 期末复习助手页：根据课程范围、剩余天数和薄弱点生成复习计划。

## 本地运行

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

## 说明

当前版本不接入真实 AI API，所有分析结果均由前端模拟规则和示例数据生成，仅用于产品原型展示。
