# BaoyanPilot 保研领航员

BaoyanPilot 是一个面向大学生的 AI 保研规划助手，旨在帮助大学生更早、更系统地进行保研准备。

项目通过个人画像评估、院校资料库、期末复习规划和保研路径建议，帮助学生发现自身优势、识别短板，并制定更清晰的保研准备路线。

## 项目背景

很多大学生并不是没有能力，而是不清楚自己是否适合保研，也不知道如何系统准备绩点、英语、科研、竞赛、学生工作和申请材料。

尤其是大一、大二阶段，很多同学对保研信息了解较少，容易出现以下问题：

- 不知道自己的当前条件是否具备保研竞争力
- 不知道绩点、英语、科研、竞赛哪个更重要
- 不知道如何选择目标院校
- 不知道夏令营、预推免、九推的区别
- 不知道如何利用期末复习稳定绩点
- 不知道如何把自己的经历整理成申请材料

因此，本项目希望通过 AI 和结构化评估，帮助大学生更早开始规划保研路径。

## 核心功能

### 1. 保研人物画像评估

用户填写个人基础信息后，系统将从以下维度进行分析：

- 绩点与专业排名
- 英语能力
- 科研经历
- 竞赛经历
- 学生工作与社会实践
- 目标院校匹配度

系统会输出：

- 综合准备度评分
- 当前人物画像
- 核心优势
- 主要短板
- 推荐准备路径
- 未来 30 天行动计划

### 2. 院校资料库

整理不同院校的保研相关信息，包括：

- 学校名称
- 学院与专业方向
- 夏令营 / 预推免 / 九推信息
- 材料要求
- 考核方式
- 适合人群
- 官网链接
- 更新时间

### 3. 期末复习规划助手

围绕保研中最重要的绩点基础，帮助学生进行课程复习规划，包括：

- 课程复习计划
- 知识点框架
- 重点题型整理
- 考前三天冲刺计划
- 一页 A4 速记版整理思路

## 技术栈

项目计划采用以下技术实现：

- React
- Vite
- Tailwind CSS
- JavaScript
- 本地 JSON 数据库
- 后续计划接入 AI API
