# BaoyanPilot 保研领航员

BaoyanPilot 是一个面向大学生的 AI 保研规划助手，旨在帮助大学生更早、更系统地进行保研准备。

项目围绕保研准备中的个人定位不清、信息获取分散、院校选择困难、绩点规划薄弱等问题，提供个人画像评估、院校资料库、期末复习规划和保研路径建议等功能，帮助学生发现自身优势、识别短板，并制定更清晰的保研准备路线。

## 在线体验

访问地址：https://baoyan-pilot.vercel.app/

## 项目背景

很多大学生并不是没有能力，也不是没有经历，而是不清楚自己是否适合保研，更不知道应该如何系统准备绩点、英语、科研、竞赛、学生工作和申请材料。

尤其是在大一、大二阶段，很多同学对保研规则了解有限，常常会遇到以下问题：

- 不知道自己的当前条件是否具备保研竞争力
- 不知道绩点、英语、科研、竞赛哪个更重要
- 不知道如何选择目标院校和专业方向
- 不知道夏令营、预推免、九推之间的区别
- 不知道如何利用期末复习稳定绩点
- 不知道如何把自己的经历整理成保研申请材料

因此，本项目希望通过结构化评估和 AI 辅助规划，帮助大学生更早开始准备保研，并逐步形成清晰、可执行的成长路径。

## 核心功能

### 1. 保研人物画像评估

用户填写个人基础信息后，系统会从多个维度进行综合分析：

- 绩点与专业排名
- 英语能力
- 科研经历
- 竞赛经历
- 学生工作与社会实践
- 实习与项目经历
- 目标院校匹配度

系统将输出：

- 综合准备度评分
- 当前人物画像
- 核心优势分析
- 主要短板识别
- 推荐准备路径
- 未来 30 天行动计划
- 风险提醒

### 2. 院校资料库

项目计划整理不同院校的保研相关信息，包括：

- 学校名称
- 学院与专业方向
- 夏令营 / 预推免 / 九推信息
- 报名时间
- 材料要求
- 考核方式
- 适合人群
- 官网链接
- 更新时间

院校资料将尽量保持结构化，并标注信息来源和更新时间，方便用户进行横向比较。

### 3. 期末复习规划助手

保研的基础是绩点，而绩点的关键在于平时学习与期末复习。本项目计划围绕课程复习场景，帮助学生生成：

- 课程复习计划
- 知识点框架
- 重点题型整理
- 考前三天冲刺计划
- 一页 A4 速记版整理思路

### 4. 保研路径建议

根据用户所处年级和当前画像，系统将提供阶段性建议：

- 大一：了解保研规则，建立绩点意识
- 大二：判断自身画像，补齐科研、竞赛、英语等短板
- 大三：准备夏令营、预推免、九推和申请材料

## 技术栈

本项目目前采用以下技术实现：

- React
- Vite
- Tailwind CSS
- JavaScript
- 本地 JSON 数据
- Vercel 部署

后续计划接入大模型 API，实现更个性化的保研分析报告生成。
